A simple assembly program

 A "Hello World" program has become the traditional first program that many people learn because of its simplicity. Our version writes "Hello!" on 8 digit displays mapped at memory addresses 0x20 to 0x27. For example if you want to show '!' in the first digit, you must write this character at address 0x20.
Tableau<-->




 This program is based on two basic concepts commonly used in programming: loops and conditional statements (cf. page 28). A loop is a sequence of instructions that is continually repeated until a condition is reached.
The program begins by initializing the address of the string and the address of the display. Then it enters a loop of loading a character from memory and displaying it until the character 0x00 (null) is found. Adresses are incremented at each step of the loop. The test is done by checking the Z flag when a new character is read from memory.
