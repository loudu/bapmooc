A simple Microprocessor

We describe here a 8-bit microprocessor for educational purpose called SRISC8. The processor runs (executes) instructions, which are stored as machine code in its main memory. It is composed of three parts: an arithmetic and logical unit, a bank of registers and a control unit.

ARITHMETIC AND LOGICAL UNIT

The ALU (Arithmetic and Logical Unit) is the central part of the processor. It takes two binary numbers as inputs, called S1 and S2, and produces an output D by applying one of its arithmetic or logical operations.





Arithmetic operations include adding two numbers, incrementing and decrementing a value. Logical operations include one's complement, logical and, inclusive and exclusive or, and left rotation.

BANK OF REGISTERS

Registers are fast access memory cells used for storing 8-bit numbers before and after computation by the ALU. There are four general-purpose registers: A, B, C and D. Register A (Accumulator) is the implicit destination register used in most operations. There are four registers for address manipulation: M1 and M2 (Memory Address), LR (Link Register) and SP (Stack Pointer) which are used for implementing function calls and returns.

CONTROL UNIT

The Control Unit reads instructions from the main memory and controls their execution. It contains two dedicated registers: PC (Program Counter) holding the address of the current instruction in main memory, and IR (Instruction Register) which contains its machine code. It also maintains binary flags that give information about the current operation: Z (Zero), C (Carry) and N (Negative). After the execution, the control unit increments the PC in order to fetch (read) the next instruction.
