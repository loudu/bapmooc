A prospective look into the future of computing

Moore's law

Moore's law is the observation, originally made by the Intel co-founder Gordon E. Moore in 1965, that over the history of computing, the number of transistors on integrated circuits doubles approximately every two years.

 This exponential improvement has dramatically enhanced the impact of digital electronics in every aspect of our society. However there is a physical limit, since this trend is based on our capacity to reduce the size of a transistor on silicon. This limit marks the end of the dominance of the single microprocessor architecture in computing. Since 2003, the new microprocessors stopped getting exponentially faster but only got 4x faster in 10 years. In the meantime, the number of cores per microprocessor has increased in order to maintain the growth in computing power. 



The basic branching operation is to go to a given address in memory.The implementation is done by loading the destination address into the PC instead of incrementing its current value.If this branching operation is combined whith a conditional statement,then the programm can select at runtime which part of the code will be executed depending of a given condition. This is the basic implementation for if-then-else and loop program structures.

Our simple microprocessor has four control flow (branching) instruction.

JPG ( Jump and Link ) assigns PC with the content of memory at PC +1 and goes to ( jump ) that location.It also saves (link) the potential return address, that is PC+2, in the Link Register (LR).

BRA ( Branch ) transfers the value of one of the address registers to the PC, thus allowing to branch to that new location.

CMP ( compare) makes a conditional test on the Zero, Carry or Negative flags.If the condition is true, then PC = PC+1 ( default behavior), else PC = PC+2.

HLT ( Halt) is a special instruction that stops the processor.

Computer Architecture-A Quantitative APr

   The first trend for increasing the processing power of a computer is to use multiple processing units instead of one. Most old and modern supercomputers are based on this idea. All new microprocessors include multiple cores instead of a single CPU. There are two basic approaches: micro-parallelism which allows to execute multiple instructions at the same time within a single processor, macro-parallelism that takes advantages of multiple processors and possibly a large number. 

Cloud computing and artificial intelligence

  Cloud computing is a synonym for distributed computing over the Internet and means the ability to run a program on a possibly huge number of computers. In the near future, there will be tiny computers everywhere constantly monitoring systems, buildings, ecosystems and even human bodies. The cloud will include intelligent agents that will help humans to access and manage this quasi-infinite infosphere. 

Promising technologies

  There is active research to explore new promising architectures and technologies: electronic nanolithography, optical and quantum computers, chemical and DNA biocomputers, synthetic neural networks and reconfigurable machines. Most of these trends are exploring the design of nanocomputers whose fundamental parts are no bigger than few nanometers.

What is a quantum computer

  A classical computer works by processing bits that represent ones and zeroes. In contrast, a quantum computer uses quantum bits (qbits) whereby information can exist as both a one and a zero or "superpositions" of the two states at the same time. Therefore, quantum computers can work in parallel rather than sequentially, allowing to solve certain problems orders of magnitude faster.
