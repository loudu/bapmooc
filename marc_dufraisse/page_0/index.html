Computer memory

At the most simplistic level, the stored program architecture of computers have three main parts: a main memory, which holds data and programs, a processor (or CPU), which processes the data by executing programs, inputs and outputs (I/O), which allow the computer to communicate with the outside world.

Memory Device

The main memory is made of an amount of bytes. For example, a 2 megabyte memory has 2 millions of bytes that can store up to 2 millions of 8-bit numbers or characters.



Memory addressing

Each byte has an address which is a number that says where the byte is in memory. The first byte has an address of 0, the next one has an address of 1, and so on. An address is also a binary number. A byte is the smallest piece of memory that can be addressed. When the computer addresses a byte in memory, it can read or write it as a whole. However, most modern computers are also able to deal with words consisting of up to 64 bits (8 bytes) or more. Inputs and outputs are generally mapped in the memory space and accessed like a memory for reading and writing.
