WHAT'S A COMPUTER?

A simple definition of a complex machine A computer is a general purpose machine that can be programmed to carry out a finite set of operations. Most computers mainly consist of a system unit, a monitor, a keyboard and a mouse, and various input/output peripherals. However, some mobile devices include new interfaces, such as a touch screen for example. 

The computer is a universal machine

The defining feature of a computer which distinguishes it from all other machines is that it can be programmed. Since the sequence of operations (the program) can be changed, a computer can theoretically solve any kind of problem for which we are able to write a program. However, in order to be really universal, the instruction set of the computer must contains all the necessary operations. A complete set of instructions includes at least arithmetic and logical operations for basic binary computation, control flow operations affecting the order of execution, and memory operations for loading and storing data.





The stored program architecture A modern computer is based on a specific model called the stored program architecture. This model is the result of the historical evolution of computing devices (cf. chapter 2). It is at least constituted of a Central Processing Unit (CPU) and a Memory. The CPU reads the instructions from the computer's memory and executes them. Data and parameters required by the program are also stored in the memory. Instead of a single CPU, recent computers use multiple core processing unit. Despite their differences, these new computers continue to follow the principle of the stored program architecture.

Hardware and software

The hardware consists of all physical devices required by the computer to work. This includes the two fundamental elements of the stored program architecture, the CPU and the Memory, but also all the other hardware parts such as a box, a power supply, a hard disk, a monitor, a keyboard, a mouse, cables, etc. The software includes all the programs required by the computer to work. This includes the Operating System and all applications such as web browsers, text editors and other tools, games, etc.

BEFORE COMPUTERS

Brief history of computers:

The history of computers begins with the need for humans to perform calculations. It has followed the increase in calculation needs and has taken advantages of progresses in other fields: mechanical clocks and automata, relays for telegraphy, vacuum tubes for detecting radio signals, transistors for radio and long distance telephony.





The first multifunction calculator:

Charles Babbage designed in 1822 a mechanical calculator called the Difference Engine (see above one of his schemas). After the attempt at making this machine fell through, he worked to design a more complex machine called the Analytical Engine. This machine marked the transition from mechanized arithmetic to general purpose computation. He was helped by Ada Lovelace, who designed the first algorithm intended to be processed by the machine. She is often considered the world's first computer programmer.
